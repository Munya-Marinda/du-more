# Du-More 📝

```React Native To-Do List App```

The React Native To-Do List App is a straightforward mobile application that allows you to create and manage your to-do tasks with ease. Whether you want to keep track of daily chores, work assignments, or personal goals, this app provides a simple and efficient solution. ✔️

## Features 🌟

### Create and Manage Tasks ✅
- Easily add new tasks with a title and description. ➕
- Organize tasks into different categories or lists for better organization. 📋
- Mark tasks as complete when you've finished them. ✅
- Delete tasks you no longer need. 🗑️

### Persistent Storage with Async Storage 📦
- Your to-do lists are stored securely using the `@react-native-async-storage/async-storage` library. 🔐
- Tasks persist even after closing the app or restarting your device, ensuring your progress is never lost. 💾

### Intuitive User Interface 📱
- The app boasts a user-friendly interface that allows you to navigate effortlessly. 🚀
- Clear visuals make it simple to distinguish between tasks and their statuses. 👁️

## Getting Started 🚀

### Prerequisites 📋
Before running the React Native To-Do List App, ensure you have the following:

- Node.js and npm installed. 📦
- React Native development environment set up. ⚛️

### Installation 💻
1. Clone this repository to your local machine. 🖥️
2. Navigate to the project folder and run `npm install` to install dependencies. 📦
3. Start the development server with `npm start` or `expo start`. 🚀
4. Follow the on-screen instructions to run the app on your preferred platform (iOS or Android). 📱

## Usage 📝
1. Launch the app, and you'll be greeted with a clean to-do list interface. 🚀
2. Add tasks by clicking the "Add Task" button and providing a title and description. ➕
3. Organize tasks into lists if desired. 📋
4. Mark tasks as complete when you've finished them. ✅
5. Delete tasks by swiping left on them. 🗑️
6. Your tasks are automatically saved and will persist across app sessions. 📦
